### Questions You Might Get

**Q: Why did you choose Vite over Create React App?**  
A: Vite provides faster development server, better build times, and is the modern standard for React projects.

**Q: How would you add a backend?**  
A: I've structured the project with a services layer (api.js) that's ready to integrate with REST APIs. Would replace local data with API calls.

**Q: How did you handle state management?**  
A: Used React hooks (useState, useEffect) for local state and localStorage for persistent data. For larger apps, would consider Redux or Zustand.

**Q: Is this mobile responsive?**  
A: Yes, mobile-first design with CSS Grid/Flexbox. Tested on multiple screen sizes.

**Q: What would you add next?**  
A: User authentication, backend API, user-submitted recipes, rating system, and PWA support.
